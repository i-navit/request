【保護者の皆様へ】
当チャンネル『ぼっちんガイド（ルミナスゲームズ）』はYouTube規約・ガイドラインを遵守し、Robloxを安心してプレイ・参加できるプラットホームを運営しております。

遊びながら学ぶ

私たちルミナスゲームズは『遊びながら学ぶ』を大切にしています。

✅ワクワクを力に：子どもたちの「好き」を原動力に、新しいことへ挑戦する心を育てます。

安心できるコミュニティ

✅プラットフォームの適正運営：ライブ配信内での安全なコミュニケーションを推奨し、不適切な言葉や行動がないか、運営チームがトラブル防止に努めます。
✅差別のないプレイ環境：年齢や性別、プレイスキルに関係なく、対等な立場で『楽しい』時間を共有します。
✅一歩を踏み出す勇気を応援：初めてでも馴染めるよう、温かい雰囲気づくりを大切にしています。「やってみたい」という気持ちを、私たちは全力で歓迎し、サポートします。

誰かの役に立つ喜び

「自分の知識が誰かの役に立つ」という成功体験を大切にしています。

✅教え合いの文化：ロブロックスを通じて得た知識を仲間に共有することで、「自分にもできる」「感謝される」という自信に繋げます。

プライバシーポリシー

当活動および本サイトでは、YouTube配信やゲームを通じて、利用者の皆様に安心して楽しんでいただけるよう、以下の通り情報の取り扱いについて定めています。

1 取得する情報と利用目的

  • YouTube上の公開情報: チャンネル名、ユーザーID、チャット情報の取得。YouTube配信をスムーズに進行するため、参加者のみなさまが楽しく過ごせる環境づくりのために利用します。
  • ゲーム内情報・スクリーンショット: ゲーム内の表示名（ニックネーム）、および一緒にプレイした際のスクリーンショット。本サイトへの掲載（活動記録）や、YouTubeコミュニティ内で利用します。

※住所・本名・電話番号等の個人情報を直接取得することはありません。サイト掲載にあたり、表示名の非表示を希望される場合はお申し出ください。

2 YouTube APIの利用について

YouTube配信においてチャット情報を取得するためにYouTube API Servicesを利用しています。取得したデータは、Googleのプライバシーポリシーに基づき適切に管理されます。

3 データの管理と第三者提供

お預かりした情報は活動の範囲内でのみ利用し、法令に基づく場合を除き、本人の同意なく第三者に提供することはありません。

4 お問い合わせ
